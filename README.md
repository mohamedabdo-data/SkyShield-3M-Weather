# 🌍 SkyShield: NASA-Powered Climate Defense System
**By Team: The Six Pyramids (ذا سكس بيراميدز)**

---

## 🛰️ رؤية المشروع / Project Vision
مشروع متكامل يهدف لحماية المدن الساحلية مثل **الإسكندرية وفينيسيا** من خطر غرق المدن الناتج عن ذوبان الجليد. يعتمد النظام على دمج تكنولوجيا الدرونز المتطورة مع بيانات الأقمار الصناعية لوكالة ناسا.

## 🏗️ النظام المعماري (The Sustainable Control Hub)
* **المبنى الزجاجي الذكي:** مركز قيادة مصمم من الزجاج الحراري المتطور، يعمل بالكامل بالطاقة الشمسية المتجددة عن طريق ألواح (Photovoltaic) على السطح.
* **غرفة التحكم:** يتم من داخلها إدارة أسراب الدرونز ومراقبة البيانات اللحظية عبر شاشات تفاعلية مرتبطة بالأقمار الصناعية.

## 🚁 نظام الدرونز (The Swarm Intelligence)
* **التحكم عن بُعد:** يتم التحكم في الدرونز عبر أنظمة اتصال (Satellite Link) من داخل المبنى.
* **خزانات المادة:** الدرونز مزودة بخزانات لمواد كيميائية صديقة للبيئة (مثل Silver Iodide) لرشها في القطبين لتقليل سرعة ذوبان الجليد.
* **تكامل بيانات ناسا:** النظام يستخدم بيانات **NASA Worldview** و **MODIS** لتحديد أماكن الرش بدقة بناءً على الخرائط الحرارية.

## 📱 تطبيق الهاتف (SkyCast App)
تطبيق تفاعلي للمواطنين يقدم:
* **Hyper-local Weather:** تنبؤات جوية دقيقة لموقعك الحالي بالثانية.
* **Climate Widget:** ويدجت تعرض لك جودة الهواء والملابس المقترحة قبل خروجك من البيت بساعة.
* **Carbon Rewards:** نظام نقاط يكافئ المستخدمين عند استخدام المواصلات العامة لتقليل الانبعاثات.

## 🛠️ التقنيات المستخدمة (Tech Stack)
- **Frontend:** HTML5, Tailwind CSS, Leaflet.js
- **Backend:** Python (NASA POWER API Integration)
- **Monitoring:** NASA EarthData (Terra, Aqua, VIIRS)
